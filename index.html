Pixel Quest To-Do — Ubah Daftar Tugasmu Jadi Petualangan Seru!

Pernah ngerasa bosan pakai aplikasi to-do list yang tampilannya itu-itu aja? Nah, Pixel Quest To-Do hadir buat ngubah cara kamu nyelesain tugas jadi lebih seru — kaya main game RPG pixel art!

Konsep Aplikasi

Pixel Quest To-Do adalah aplikasi to-do list berbasis web (dibuat dengan Flutter) yang dikemas dengan gaya RPG. Kamu bisa memilih karakter emoji favoritmu, lalu setiap tugas yang berhasil diselesaikan akan menambah XP untuk naik level. Jadi, bukan cuma checklist biasa — tapi benar-benar terasa seperti petualangan kecil setiap hari!

  • Pilih Karakter mulai dari penyihir, elf, astronot, dan lainnya.
  • Buat dan kelola misi harian Tugas tugas yang ingin kamu selesaikan.
  • Dapatkan XP Setiap kali kamu menyelesaikan misi(tugasmu).
  • Naik Level Naik Level setiap XP penuh.
  • Data Tersimpan di local storage browser.
  • Tema pixel RPG gelap biar vibes-nya kaya game retro.

Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan:

  • Flutter Web untuk membuat aplikasi cross-platform.
  • dart:html menyimpan data ke localStorage.
  • Material Design untuk tampilan yang rapi dan responsif.
  • Naik Level Naik Level setiap XP penuh.

Cara Kerja Singkat

  1. Pilih karakter emoji favoritmu.
  2. Tambahkan tugas baru lewat kolom input.
  3. Centang tugas setelah selesai → kamu dapat XP!
  4. Kalau XP sudah cukup, karaktermu naik level otomatis.
  5. Semua data tersimpan di browser, jadi kamu gak perlu login apa pun.

Fitur Cerdas

Aplikasi ini juga punya logika agar XP tidak bisa dieksploitasi. Kalau kamu uncheck tugas yang sudah selesai, XP kamu otomatis berkurang. Artinya, kamu cuma dapat XP sekali per tugas yang benar-benar diselesaikan — adil dan realistis!

Tampilan & Desain

Tema utama menggunakan palet warna gelap dengan sentuhan cyan futuristik untuk efek RPG modern:

  • #1A1A1A (background)
  • #393E46 (card)
  • #00ADB5 (accent)
  • #FFFFFF (text)

UI-nya dibuat simpel tapi tetap ada nuansa game, lengkap dengan emoji karakter besar di bagian atas dan progress bar XP di bawahnya. Semua terasa ringan karena berjalan langsung di browser.

Kesimpulan

Pixel Quest To-Do bukan sekadar aplikasi catatan tugas — tapi juga cara baru untuk memotivasi diri sendiri. Setiap checklist yang kamu buat bukan cuma kerjaan yang selesai, tapi juga langkah menuju level baru!

Ingat: setiap hari adalah petualangan baru, dan kamu adalah hero-nya
